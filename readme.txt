== Description ==

This WordPress plugin automatically disables the 'Choose a Pattern' popup in the new editor when creating pages in the TT4 Theme.

== Frequently Asked Questions ==

= How does this plugin work? =

This plugin enqueues a JavaScript file that simulates a click on the close button of the 'Choose a Pattern' popup, preventing it from appearing.

= Is this compatible with other themes? =

This plugin is specifically designed for the TT4 Theme. Compatibility with other themes may vary.
